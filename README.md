# APEX TECH - Student Tech Bootcamp Website

A simple, fast, and easy-to-deploy website for APEX TECH bootcamp targeting students in grades 8-10.

## Features

- 📱 Fully responsive design
- ⚡ Pure HTML/CSS/JavaScript (no build tools needed)
- 🎨 Modern, professional design with animations
- 📊 Scroll animations and interactive elements
- 🚀 Easy deployment on GitHub Pages

## Files

- `index.html` - Main website structure
- `styles.css` - All styling and animations
- `script.js` - Interactive features and animations

## Deployment on GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select `main` branch as source
4. Your site will be live at `https://yourusername.github.io/apex-tech-bootcamp`

## Customization

### Change Logo
Replace the placeholder logo URL in `index.html`:
```html
<img src="https://via.placeholder.com/120x60?text=APEX+TECH" alt="APEX TECH Logo" class="logo">
```

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0A1428;
    --accent-color: #00D9FF;
    --text-color: #333;
}
```

### Add Contact Form
Uncomment the contact section in `index.html` and add your form handling.
